# To-Do List

Manage your tasks and assignments with the powerful StudyPlus to-do system.

## Creating Tasks

### Manual Tasks

1. Click **+ Add Task** in the To-Do List
2. Enter a title and optional description
3. Set a due date (optional)
4. Choose a priority level
5. Click **Save**

### AI-Generated Tasks

The **AI To-Do Generator** can break down complex assignments:

1. Go to **To-Do Generator** in the sidebar
2. Paste your assignment description
3. Click **Generate Tasks**
4. Review and add tasks to your list

:::tip Pro Tip
The AI is great at breaking down essay assignments, research projects, and multi-step homework into actionable chunks.
:::

## Task Properties

Each task can have:

| Property | Description |
|----------|-------------|
| **Title** | Short description of the task |
| **Description** | Detailed notes or instructions |
| **Due Date** | When the task should be completed |
| **Priority** | High, Medium, or Low |
| **Status** | Not started, In progress, Completed |

## Managing Tasks

### Completing Tasks

- Click the checkbox to mark as complete
- Completed tasks move to the "Done" section

### Editing Tasks

- Click on a task to open the edit panel
- Make changes and click **Save**

## Filtering & Sorting

Use the toolbar to:

- **Filter by**: Status, Priority, Due date
- **Sort by**: Date created, Due date, Priority
- **Search**: Find tasks by title or description

## Synced vs Manual Tasks

- **Synced tasks** (from eClass) have a link icon
- **Manual tasks** are created by you
- Both types can be edited and completed
