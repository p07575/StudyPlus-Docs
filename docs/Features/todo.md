# To-Do List

Manage your tasks and assignments with the StudyPlus to-do system.

## Accessing To-Do List

Navigate to **To-Do** in the sidebar to view and manage your tasks.

## Creating Tasks

### Manual Tasks

1. Type your task in the input field at the top
2. Press **Enter** or click **Add** to create the task

### AI-Generated Tasks

The **AI To-Do Generator** can break down complex assignments:

1. Go to **SAILS** ‚Üí **To-Do List Generator**
2. Describe your assignment or project
3. Click **Generate**
4. Review the generated tasks with priorities
5. Edit tasks if needed
6. Add selected tasks to your list

:::tip Pro Tip
The AI is great at breaking down essay assignments, research projects, and multi-step homework into actionable chunks with priority levels.
:::

## Managing Tasks

### Completing Tasks

- Click the **checkbox** next to a task to mark it complete
- Completed tasks show with a strikethrough

### Reordering Tasks

- Use the **up/down arrows** to reorder your tasks
- Organize by priority or due order

### Deleting Tasks

- Click the **delete button** (üóëÔ∏è) on a task to remove it

## Task Properties

Each task has:

| Property | Description |
|----------|-------------|
| **Title** | The task text |
| **Completed** | Whether the task is done |
| **Order** | Position in your list |

## AI-Generated Task Properties

Tasks from the AI generator include:

| Property | Description |
|----------|-------------|
| **Title** | The task description |
| **Priority** | High, Medium, or Low |

You can edit the generated tasks before adding them to your list.
